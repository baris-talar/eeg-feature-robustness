EEG CROSS-DATASET RESEARCH — DATASET NOTES

==================================================
DATASETS
==================================================

PhysioNet EEG Motor Imagery
- Subjects: 109
- Sampling rate: 160 Hz
- Channels: 64 EEG (→ reduced to 22)
- Tasks: left, right, feet, tongue, rest
- Relevant runs: 4, 8, 12 (left/right only)
- Labels (MNE): left_hand, right_hand, rest
- Event codes: T1=left, T2=right, T0=rest

BCI Competition IV 2a (BNCI2014_001)
- Subjects: 9
- Sampling rate: 250 Hz
- Channels: 22 EEG (+ EOG ignored)
- Tasks: left, right, feet, tongue
- Labels (MNE): left_hand, right_hand, feet, tongue
- Event codes: 769=left, 770=right, 771=feet, 772=tongue

==================================================
SHARED CHANNEL SPACE (22 EEG)
==================================================

Fz,
FC3 FC1 FCz FC2 FC4,
C5 C3 C1 Cz C2 C4 C6,
CP3 CP1 CPz CP2 CP4,
P1 Pz P2 POz

- PhysioNet → reduce to these 22
- BCI2a → select these 22 (drop EOG)

==================================================
TASK & LABELS
==================================================

Binary classification: LEFT vs RIGHT

Removed:
- PhysioNet: rest
- Both: feet, tongue

Label encoding:
- left_hand  → 0
- right_hand → 1

==================================================
PREPROCESSING PIPELINE
==================================================

Applied identically to both datasets:

1. Select 22 shared channels
2. Resample to 250 Hz
3. Bandpass filter: 4–40 Hz
4. Epoch extraction:
   - window: 0.5–2.5 s post-cue
5. Keep left/right trials only
6. Convert to NumPy arrays

Output:
- X: (n_trials, 22, 501)
- y: (n_trials,) ∈ {0,1}

==================================================
FEATURE SETS
==================================================

A — Time (baseline)
- Mean + variance per channel
- Dim: 44
- Note: mean ≈ 0 after filtering

B — Band power
- Welch PSD, 4 bands: delta, theta, alpha, beta
- Log transform applied
- Dim: 22 × 4 = 88

C — FFT magnitude
- FFT in 4–40 Hz
- Interpolated to fixed bins (n_bins=36)
- Dim: 22 × 36 = 792
- No log transform

==================================================
KEY DIFFERENCES (DISTRIBUTION SHIFT)
==================================================

- Subjects: 109 vs 9
- Sampling rate: 160 → 250 vs 250
- Channels: 64 → 22 vs 22
- Structure: mixed runs vs structured sessions

→ Causes cross-dataset generalisation difficulty

==================================================
EXPERIMENTAL CONSTRAINTS
==================================================

1. Same channels (22)
2. Same labels (binary)
3. Same preprocessing
4. No data leakage
5. Cross-dataset evaluation:
   - train on A → test on B
